## P1 / THE PRODUCTIVE ACT

**PRODUCTIVE RETURN (U7)** — B(0,P1). Type D. FORCED.
Self-action that generates genuinely new content. R²=R+I: the return exceeds the departure by the identity. Instances: Fibonacci growth, K6' loop, SIL discovery cycle.
*Use when self-action generates content. Do not conflate with cyclic return (J²=I, generates nothing new).*

**RECURSIVE COMPLETION (U2)** — B(0,P1). Type D. FORCED.
Closure that simultaneously terminates and feeds the next level. Phase 1: stabilize. Phase 2: the stabilized object becomes input. K6' closes → K7' encodes. SIL-1 stabilizes → discovery operator acts.
*Use when closure at level n feeds level n+1. Distinguish from terminal closure (no further reentry).*

**CLOSURE** — B(2,P1). Type C (contranym). FORCED.
Self-action stabilization across four strata: closure mode (trajectory character), closure type (terminal/recursive/boundary — three projections via central collapse), closure loop (K6'/K7'/K8), closure status (prediction chain grade). The contranym: terminal (P1, process finishes) vs gateway (P2, stabilized object enters next level). See SEMANTICS §1 C1.
*Always type as mode/type/loop/status. Never bare "closure" in theorem statements.*

**FORCING** — B(2,P1). Type A (synonym cluster). FORCED.
The property of zero-branching derivation: the conclusion is unique given the premises. Synonym cluster: "forced," "unique," "zero-branching," "derived" (in strict sense), "the unique survivor," "absolutely forced." See GOVERNANCE §2 for G.1–G.9 disambiguation.
*Reserve "forced" for G.1 (strict, br_s=0) in theorem statements. Other generation classes use specific names.*

**CANONICAL** — B(2,P1). Type A. FORCED.
Two strengths. Strong-canonical = forced (no alternative exists, G.1). Weak-canonical = naturally preferred (Dist-ward functor is natural, Co-Dist-ward is not). See SEMANTICS §5 Principle 6.
*Specify which strength. "Canonical" alone is ambiguous.*

**COMPRESSION** — B(3,P1). Type C (contranym). FORCED.
Information reduction that is simultaneously invariant strengthening. The contranym: loss (fewer states) vs clarity (stronger structure). See SEMANTICS §1 C9.
*Use when information decreases and structure increases simultaneously.*

**COMMITMENT** — B(0,P1). Type D. FORCED.
Irreversible stabilization from repeated Möbius contraction at rate φ̄² per step (SUBSTRATE §14¾). After m steps: residual ≤ φ̄^{2m}. At Level 5: each K6' pass IS one commitment increment.
*Use for irreversible convergence toward an attractor. Distinguish from naming (one pass) and world-formation (sustained across spatial domain).*

**SELF-SIGNATURE** — B(4,P1). Type A. FORCED.
σ = (1/2, φ̄/2, φ̄²/2). The framework's computational identity. Unique from GPF (ALGEBRA §11). σ₁ = 1/2 = ∫_{P3}α (convergence witness). The observer's preserved invariant across recursive negation layers.

**φ̄ (PHI-BAR)** — B(3,P1). Type A. FORCED.
The golden ratio conjugate (√5−1)/2 ≈ 0.618. The Möbius attractor. The co-closure of the stance grammar at Level 3. The contraction rate per step is φ̄².

## P2 / THE MEDIATING ACT

**POISED SYMMETRY (U5)** — B(0,P2). Type D. ENCODED.
Balanced state that is maximally generative. All channels open, no preferential allocation. ρ = 1/2 (self-referential neutral point). The phase-neutral substrate before asymmetry develops.

**ADMISSIBLE MINIMALITY (U6)** — B(0,P2). Type D. FORCED.
Coincidence of smallest-sufficient with largest-necessary. S₀ = {0,1} is the unique minimal nontrivial binary seed. The quotient q as unique minimal idempotent (CATEGORY §6).

**MINIMAL** — B(0,P2). Type C (contranym). FORCED.
Least/smallest AND maximally generative. The contranym: reduction (fewer elements) vs generation (the minimal seed produces everything). See SEMANTICS §1 C5.

**THRESHOLD** — B(4,P2). Type C (contranym). FORCED.
Barrier AND entry point. The contranym: bound (cannot pass) vs initiation (the crossing begins something new). φ̄² as the four-domain threshold (P1_PRODUCTION §5). See SEMANTICS §1 C6.

**SCALE** — B(5,P2). Type C. FORCED.
Two opposed readings: scale-as-capacity (P1: more states, more resolution) vs scale-as-limitation (P3: finite observer, constitutive blind spot). Every conscious observer lives in this tension (OBSERVER §5 Thm 10½.22).

**EMERGENCE** — B(all,P2). Type A. FORCED.
In the framework: forced derivation from prior structure, zero branching. NOT philosophical novelty. See SEMANTICS §5 Principle 5.
*Replace "emerges" with "derives" in formal contexts.*

**DERIVE** — B(all,P2). Type A. VARIES.
Superverb — does not by itself imply G.1 or FORCED. Must carry its production subtype: strictly forced (G.1), observer-forced (G.6), transport-derived (G.7), reconstruction-derived (G.8), candidate-derived (G.5/RESONANT). See GOVERNANCE §2.
*Never use bare "derived" in commitment-bearing prose.*

**PREDICTION** — B(6,P2). Type A. VARIES.
Four subtypes: theorem-prediction (FORCED, zero-branching physical statement), structural prediction (ENCODED/RESONANT, strong fit without full closure), processed prediction (framework inputs + standard external tools), speculative (multiple proposals, unresolved). See PHYSICS §5.

**L (LOG₂(φ))** — B(4,P2). Type A. FORCED.
≈ 0.694. Appears in four domains: Landauer cost 1/L (P2_MEDIATION §4), Cosmological Tower Equation coefficient (PHYSICS §7), two-axis model 2L bits per K6' pass (OBSERVER §5), information budget L content / (1−L) overhead. One constant, four roles.

## P3 / THE OBSERVER ACT

**OCCLUSIVE DISCLOSURE (U1)** — B(2,P3). Type D. FORCED.

**CONSTITUTIVE OCCLUSION (U4)** — B(5,P3). Type D. FORCED.

**CANONICAL EXTRACTION (U8)** — B(2,P3). Type D. FORCED.

**OBSERVATION** — B(2,P3). Type C (contranym). FORCED.

**BLINDNESS** — B(5,P3). Type C (contranym). FORCED.

## CROSS-PROJECTION

**SELF-RELATING DIFFERENCE (SRD)** — B(0,all). Type A. FORCED.
The minimal operational compression of the co-primitives P.1+P.2. A Dist endomorphism on |D|≥2 whose iterates are well-defined. Acts on itself. Everything in the framework is SRD at some tower depth through some projection face.

**R(R) = R** — B(all,all). Type A. FORCED.

**PHASE** — B(0,all). Type C. FORCED.
Two distinct uses: phase as ρ-parameter (continuous, ρ∈[0,1], Phase-Dist) vs phase as qualitative regime (compressive/expansive). See SEMANTICS §5 Principle 4.
*Use "ρ" for the parameter, "regime" for the qualitative state.*

**NEUTRALITY** — B(0,P2). Type C (contranym). ENCODED.
Inert balance AND maximum generativity. ρ=1/2: zero preferential allocation, all channels open. See SEMANTICS §1 C7.

**CONVERGENCE WITNESS** — B(all,all). Type A. FORCED.
Two independent derivation routes from different projection faces producing the same result, with convergence forced by CATEGORY Thm 4.3 (three simultaneous readings). Five instances: ρ-regulation (SUBSTRATE §14½), Dist uniqueness (CATEGORY §4), e (P2_MEDIATION §1), σ₁=∫_{P3} (P1_PRODUCTION §2), Bekenstein (OBSERVER §2).

**RECURSIVE DISCLOSURE** — B(5, P3→P1). Type D. FORCED.

**|e⟩⟨π| (CROSS-SECTOR OPERATOR)** — B(4,P2↔P3). Type D. FORCED.
The rank-1 nilpotent representing the forbidden crossing between P2 and P3 sectors. |e⟩⟨π|² = 0 (mode (iii): from B(h,N) = 0, Killing orthogonality). tr = 0, NOT Hermitian.
